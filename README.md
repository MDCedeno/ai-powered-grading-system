## 🚀 Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/ai-powered-grading-system.git
   cd ai-powered-grading-system
   ```

2. **Set up XAMPP:**
   - Install XAMPP from [apachefriends.org](https://www.apachefriends.org/)
   - Start Apache and MySQL services

3. **Configure the project:**
   - Copy the project to your XAMPP htdocs directory:
     ```bash
     # On Windows
     copy ai-powered-grading-system C:\xampp\htdocs\

     # On macOS/Linux
     cp -r ai-powered-grading-system /Applications/XAMPP/htdocs/
     ```

4. **Database Configuration:**
   - Open `ai-powered-grading-system/backend/config/db.php`
   - Update database credentials as needed

## 🗄 Database Setup

1. **Create Database:**
   ```sql
   CREATE DATABASE your_database_name;
   ```

2. **Import Schema:**
   - Create the necessary tables (users, courses, grades, etc.)
   - Set up foreign key relationships

3. **Run Seeders:**
   ```bash
   cd ai-powered-grading-system/backend/database/seeders/
   php UserSeeder.php
   ```

## 📖 Usage

1. **Access the Application:**
   - Open your web browser
   - Navigate to: `http://localhost/ai-powered-grading-system/frontend/views/login.php`

2. **User Roles:**
   - **Super Admin**: Full system administration
   - **MIS Admin**: User management and system monitoring
   - **Professor**: Course management and student grading
   - **Student**: Grade viewing and quiz participation

3. **Explore Features:**
   - **Students:** View grades, take quizzes, track progress
   - **Professors:** Manage classes, enter grades, generate reports
   - **Admins:** Monitor system, manage users, configure settings
