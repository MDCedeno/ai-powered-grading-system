## ğŸš€ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/ai-powered-grading-system.git
   cd ai-powered-grading-system
   ```

2. **Set up XAMPP:**
   - Install XAMPP from [apachefriends.org](https://www.apachefriends.org/)
   - Start Apache and MySQL services

3. **Configure the project:**
   - Copy the project to your XAMPP htdocs directory:
     ```bash
     # On Windows
     copy ai-powered-grading-system C:\xampp\htdocs\

     # On macOS/Linux
     cp -r ai-powered-grading-system /Applications/XAMPP/htdocs/
     ```

4. **Database Configuration:**
   - Open `ai-powered-grading-system/backend/config/db.php`
   - Update database credentials as needed
